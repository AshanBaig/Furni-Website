==============================================================
Welcome to our Online Shopping Store Application
==============================================================
1. Overview:

   - This application is built using Flask and provides functionalities for an online shopping store.
   - Users can sign up, log in, browse products, add items to their cart, and proceed to checkout.

2. Prerequisites:

To run our program, please import the following prompt in your terminal:
pip install flask 
pip install flask_mail
Run the main file ‘classes.py’ to start the program.
After installing these, copy the following link from your terminal to your browser: http://127.0.0.1:5000
Note: 
Make sure your internet connection is turned ON.
Use valid email IDs with ‘@gmail.com’ to receive emails. 
Don’t use ‘@’ other than with gmail.com and special characters.

4. Features:

- User Authentication: Secure login and registration with hashed passwords.
- Product Management: Admin panel for adding and updating products.
- Shopping Cart: Dynamic cart management with real-time updates.
- Order Processing: Checkout and confirmation emails.
